# 文件分享系统
## 项目简介
这是一个简单的文件分享系统，允许用户上传文件并生成下载链接。系统支持以下功能：

- 文件上传与临时存储
- 生成唯一下载链接
- 设置下载次数限制
- 自动清理过期文件


## 使用说明
### 启动程序
```
python app.py
```

- 访问首页 `/`
- 上传 `/admin/add`
